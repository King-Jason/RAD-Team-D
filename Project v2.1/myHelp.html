This is the help page for the user.


This webpage is created to store movies collection.
In the index page or in the main page called 'Home', there will be three sections in the page.
First of all, there is a section to search for a movie. The user can search for the movies based on the title,
genre, rating, and year and if the searched information is found, it will outputting the full details in table,
if not found, a user friendly webpage will show up.

Second of all, there is a button to show all of the movies collection that we have.
It will outputting the full details about the movies including, id, title, studio, status,
sound, versions, recretprice, rating, year, genre, and aspect.

Third of all, there is a button to show top 10 most frequently search movies.
This button will outputting a bar chart about the top 10 most frequently search movies.
It will outputting the name and the number of how many times the movie has been searched.
The bar chart will outputting the movies from the most searched ones to the least searched ones.
